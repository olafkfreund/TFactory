# Social post templates

Channel-adapted copy for the "tests that refuse to lie" showcase, drawn from the
blog post at `docs/_posts/2026-07-19-tests-that-refuse-to-lie.md`. `reddit-post.md`
leads hard with the never-overclaim story (the verifier capping a build at VAL-0
over one failing unicode edge case) and carries a short FAQ for r/programming,
r/QualityAssurance, and r/devops; `linkedin-post.md` is the professional cut on
the testing-integrity angle with a hashtag block. All copy is understated,
technical, and honest about the one gap the live run surfaced — edit voice and
length to fit before posting, and keep the facts as written.
