# Code of Conduct

This project adopts the **Contributor Covenant, version 2.1**.

The full text is available at:
https://www.contributor-covenant.org/version/2/1/code_of_conduct/

A plain-text copy:
https://www.contributor-covenant.org/version/2/1/code_of_conduct/code_of_conduct.txt

## Reporting

To report a violation, use **GitHub's private reporting** channel — see
[SECURITY.md](SECURITY.md) for the link. Reports are seen only by
project maintainers. Retaliation against reporters is itself a
violation.

## Enforcement

Maintainers will review reports privately and respond within 7 days.
Outcomes may include private warning, temporary or permanent ban from
project spaces, or removal of contributions, following the
Enforcement Guidelines in the Contributor Covenant.
